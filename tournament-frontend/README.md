# Tournament Hub - Student Competitions Platform

Tournament Hub is a modern web platform that aggregates student competitions across Asia-Pacific, starting with debate tournaments. Our mission is to make competitions more accessible by solving the problem of fragmented and scattered information.

## Features

- Browse and search tournaments by type, category, and location
- Create and manage tournament listings
- User profiles with registered and organized tournaments
- Responsive design for all devices
- Modern UI with Material-UI components

## Tech Stack

- React 18
- TypeScript
- Material-UI (MUI)
- React Router
- Vite
- Redux Toolkit (for state management)

## Getting Started

### Prerequisites

- Node.js 16.x or later
- npm 7.x or later

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/tournament-hub.git
cd tournament-hub
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
